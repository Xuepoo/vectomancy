# Vectomancy v1.0 发布说明

我们很高兴发布 Vectomancy 的第一个版本。此版本标志着我们在将图形文件转换为数学参数方程和渲染脚本方面迈出了重要一步。

## 主要成就

- 解决 AST 爆炸问题：通过 Zlib 和 Base64 编码，有效解决了大型图形文件导致的 AST 爆炸问题。
- 优化平滑处理：引入了 Spline 和 Chaikin 平滑算法，显著减轻了图形转换过程中的阶梯状和锯齿状伪影。
- 增强配置灵活性：增加了简化配置选项，用户现在可以通过 --tolerance 和 --min-path-len 参数精确控制输出结果。
- 广泛的格式支持：全面支持多种数学软件和文档格式，包括 GeoGebra (.ggb), Kmplot (.fkt), HTML canvas, LaTeX, Wolfram, Python 和 JSON。
- 提升 IDE 性能：自动在所有输出文件中注入 LSP 忽略注释，彻底解决了大型生成文件导致的 IDE 卡顿问题。
- 稳定性验证：完成了针对 30 多张复杂图像在所有模式和格式下的全面稳定性测试。

## 未来规划

- 性能提升：引入 GPU 加速和多线程处理，进一步提高转换速度。
- 功能扩展：增加对彩色绘图的支持。
- 生态集成：集成更多数学软件，如 Desmos, Julia, Matlab 等。
